# PyChessDesktop - Leia-me

## Descrição
PyChessDesktop é um jogo de xadrez com interface gráfica, puzzles, sons e skins personalizáveis, desenvolvido em Python.

## Requisitos
- Python 3.10 ou superior
- Dependências listadas em `requirements.txt`

## Instalação
1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Clayton8240/PyChessDesktop.git
   cd PyChessDesktop
   ```
2. **Crie um ambiente virtual (opcional, mas recomendado):**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. **Instale as dependências:**
   ```bash
   pip install -r requirements.txt
   ```

## Execução
Execute o programa principal:
```bash
python3 main.py
```

## Observações
- Certifique-se de que a pasta `assets/` e seus subdiretórios estejam presentes, pois contêm imagens, sons e fontes necessários.
- Para melhor experiência, execute em um ambiente gráfico (desktop Linux).
- Caso encontre problemas de fontes ou sons, verifique se as dependências do sistema para SDL2 e fontes estão instaladas (ex: `libsdl2-dev`, `libsdl2-mixer-2.0-0`, `fonts-dejavu`).
